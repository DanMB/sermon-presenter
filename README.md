# ourpresenter

Built in Svelte 5 with Typescript and SCSS.
Native app made with [Tauri](https://tauri.app/)

## SCRIPTS

###### Start the dev server and run in Tauri:

- `pnpm start`

###### Compile a Tauri build for distribution:

- `pnpm dist`

###### Compile a web client build for serving:

- `pnpm build`
